# git
1. ls -al ~/.ssh 检查有无ssh密钥

2. ssh-keygen -t rsa -C "your_email@example.com" 创建密码

3. git add 文件上传到缓存区

4. git commit 提交文件 

5. git remote add origin 仓库地址  建立远程连接

6. git push -u origin master第一次推送master分支的所有内容；

7. git remote -v 查看远程仓库
